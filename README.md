# RAG LangChain Lab

LangChain 기반 RAG 시스템을 학습하고 실험하기 위한 개인 레포지토리입니다.

이 저장소는 인프런의
“RAG를 활용한 LLM Application 개발 (feat. LangChain)”
강의 내용을 바탕으로 정리한 학습 아카이브와,
개인적인 실험 및 확장 내용을 함께 담고 있습니다.

## Goals

- RAG 파이프라인 구조 이해
- LangChain 기반 문서 검색/응답 흐름 학습
- Chroma, Pinecone 등 벡터 DB 활용 방식 비교
- PromptTemplate, retrieval 전략, evaluation 실습

## Structure

- `course-archive/`: 강의 기반 실습 노트북
- `experiments/`: 추가 실험
- `notes/`: 학습 정리
- `data/`: 실습용 데이터
